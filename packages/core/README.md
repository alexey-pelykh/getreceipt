# @getreceipt/core

[![npm](https://img.shields.io/npm/v/%40getreceipt%2Fcore?logo=npm)](https://www.npmjs.com/package/@getreceipt/core)
[![License: AGPL-3.0-only](https://img.shields.io/npm/l/%40getreceipt%2Fcore)](https://github.com/alexey-pelykh/getreceipt/blob/main/LICENSE)

The engine for [getreceipt](https://github.com/alexey-pelykh/getreceipt) — the source-adapter registry and resolver that the CLI and MCP server build on.

> **Unofficial.** Not affiliated with, endorsed by, or supported by any of the services it integrates with. See the [project README](https://github.com/alexey-pelykh/getreceipt#readme) for the full disclaimer.

> **Status: `0.1.0` scaffold.** No product logic yet — source adapters, auth, and resolution land in later issues. The published API surface is intentionally minimal and will change before a stable release.

## Install

```sh
npm install @getreceipt/core
```

Requires **Node.js ≥ 22.19.0**, ESM only.

Most users want the [`getreceipt`](https://www.npmjs.com/package/getreceipt) umbrella (CLI + MCP server), not this library directly.
